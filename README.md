# 클라우드 연동 가계부 앱

# 1. 개요
Firebase 기반 실시간 동기화를 지원하는 가계부 앱으로, 수입·지출 기록, 통계 분석, 목표 저축 기능을 제공하며 친구 초대와 공유 목표 방 생성을 통해 여러 사용자가 공동 목표를 함께 관리할 수 있다.

# 2. 개발 환경
- IDE : Android Studio (2025.01.04 Build)
- Language : Kotlin
- UI Framework : Jetpack Compose
- Minimum SDK : API 26 (Android 8.0)
- Emulator : Pixel 9 Pro XL (Android 16, API 36)
