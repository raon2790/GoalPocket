# 클라우드 연동 가계부 앱

# 1. 개요
Firebase 기반 실시간 동기화를 지원하는 가계부 앱으로, 수입·지출 기록, 통계 분석, 목표 저축 기능을 제공하며 친구 초대와 공유 목표 방 생성을 통해 여러 사용자가 공동 목표를 함께 관리할 수 있다.

# 2. 기대 효과
- 소비 성향과 지출 패턴을 시각적으로 분석 가능
- 목표 저축의 진행 상황과 달성률을 직관적으로 확인 가능
- 개인 중심의 가계부에서 공동 재무 관리 도구로 확장 가능

# 3. 개발 환경
- IDE : Android Studio (2025.01.04 Build)
- Language : Kotlin
- UI Framework : Jetpack Compose
- Minimum SDK : API 26 (Android 8.0)
- Emulator : Pixel 9 Pro XL (Android 16, API 36)

# 4. 기능

## 4.1. 기록 및 관리

- 수입/지출 내역 기록
- 캘린더 형식으로 내역 확인
- 기록 수정/삭제
- 정기 결제 등록 : 알림 및 월 지출 자동 반영

## 4.2. 검색 및 분류

- 검색/필터 : 기간, 카테고리, 금액 범위, 제목/메모 키워드
- 정렬 : 날짜, 금액, 카테고리 기준 오름차/내림차순
- 카테고리 기반 분류 및 통계 활용

## 4.3. 분석 및 목표 관리

- 월별 합계/잔액 확인
- 카테고리별 지출 그래프
- 전월 및 지난해 대비 비교
- 목표 저축 : 목표 금액/기한 설정, 잔액 기반 달성률 계산

## 4.4. 공유 기능

- 친구 추가 및 관리 : 초대 코드를 통한 친구 등록 및 목록 관리
- 공유 목표 : 여러 사용자가 함께 보는 목표 방 생성 및 진행 상황 공동 확인

# 5. UX/UI

- 시스템 테마(라이트/다크모드) 자동 적용
- 화면 회전 대응
- 해상도와 종횡비에 따른 반응형 UI 구성

# 6. DB 연동

- Firebase Authentication + Cloud Firestore 사용
- Google 로그인으로 사용자 식별 후 사용자별 데이터 분리
- 여러 기기에서 동일 계정 로그인 시 실시간 데이터 동기화
- 개인 데이터는 사용자별 경로에 저장
- 공유 목표는 별도의 공유 컬렉션에서 관리
- 원시 거래 데이터는 Cloud Firestore에 저장하고, 통계 및 비교 계산은 클라이언트에서 처리

# 7. 보안

- Firebase Authentication 기반 사용자 인증
- Firestore Security Rules 적용
    - 개인 데이터는 본인 계정(uid)에 대해서만 접근 가능
    - 공유 목표는 해당 목표의 멤버로 등록된 사용자만 접근 가능
- 앱 잠금 기능 (생체인식, PIN)
- 숨김 카테고리 열람 시 추가 인증 요구
